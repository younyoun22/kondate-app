# NINTENDO風 こんだて計画

夫婦二人暮らしの夜ご飯献立(月〜金+任意で土日)を提案するWebアプリ。

- 見た目: Nintendo.com 2001年風デザイン(スマホ用1カラム)
- 献立: 内蔵レシピ約200品からシャッフル生成(通信・課金なし)
- お楽しみ: PokeAPIで「今日の相棒ポケモン」を表示
- 各料理の「レシピをみる」リンクから作り方の検索結果を開ける

## 使い方

GitHub Pagesで公開されたURLをスマホのブラウザで開き、「こんだてを つくる!」を押す。
iPhoneは共有メニューから「ホーム画面に追加」するとアプリのように使える。

## ファイル構成

- `index.html` — アプリ本体(HTML/CSS/JS)
- `recipes.js` — レシピデータ(主菜/副菜/汁物)。レシピを増やすときはこのファイルに追記する

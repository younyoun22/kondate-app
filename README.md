# NINTENDO風 こんだて計画

夫婦二人暮らしの夜ご飯献立(月〜金+任意で土日)を提案するWebアプリ。

- 見た目: Nintendo.com 2001年風デザイン(スマホ用1カラム)
- 献立: 内蔵レシピ約200品からシャッフル生成(通信・課金なし)
- 各料理は2人分の分量つき、買い物リストで食材ごとに量を集計表示
- おかいものリストは食材ごとにチェックを付けられる(調味料の在庫確認欄つき)
- 気に入った日は「キープ」で固定し、他の曜日だけ引き直せる
- 「きょうの ぶんだけ」ボタンで1食分だけサクッと提案(1食モード)
- 週をまたいで生成しても、直近の献立と同じ料理が連続で出にくいよう重複を回避
- お楽しみ: PokeAPIで「今日の相棒ポケモン」を表示
- 各料理の「レシピをみる」リンクから作り方の検索結果を開ける
- PWA対応: ホーム画面に追加してアプリのように起動、オフラインでも直近の献立を表示

## 使い方

GitHub Pagesで公開されたURLをスマホのブラウザで開き、「こんだてを つくる!」を押す。
iPhoneは共有メニューから「ホーム画面に追加」するとアプリのように使える(オフラインでも起動可能)。

## ファイル構成

- `index.html` — アプリ本体(HTML/CSS/JS)
- `recipes.js` — レシピデータ(主菜/副菜/汁物)。レシピを増やすときはこのファイルに追記する
- `manifest.json` — PWA用マニフェスト(アプリ名・アイコン・テーマカラーなど)
- `sw.js` — Service Worker。index.html/recipes.js/manifest.jsonはnetwork-first、アイコン画像はcache-firstでキャッシュし、オフライン時もアプリを起動できるようにする
- `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` — PWA/ホーム画面用アイコン
